# Changelog

All notable changes to this project will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [1.0.0] - 2024-01-XX

### Added
- Initial release of 100xdevs Web Wrapper
- Multi-platform support (Windows, Linux, macOS)
- Tabbed interface for Projects and App URLs
- Navigation controls (back, forward, reload)
- Dark and light theme support with persistence
- System tray integration with quick navigation
- Security features with trusted origins and OAuth support
- Auto-update functionality via GitHub Releases
- Custom protocol handler skeleton
- GitHub Actions CI/CD for automated builds
- Comprehensive documentation and contributing guidelines

### Security
- Context isolation enabled
- Node integration disabled in renderer
- Permission requests restricted to trusted origins
- External link handling with system browser
- OAuth redirect detection and handling
