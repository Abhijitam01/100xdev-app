# 100xdevs Web Wrapper

[![License: MIT](https://img.shields.io/badge/License-MIT-yellow.svg)](https://opensource.org/licenses/MIT)

Open source Electron app for Windows, Linux, and macOS wrapping 100xdevs Projects and App.

## Download

[GitHub Releases](https://github.com/Abhijitam01/100xdev-app/releases)

## Features

- Multi-platform support (Windows, Linux, macOS)
- Tabbed interface
- Dark/light theme
- System tray integration
- Auto-updates via GitHub Releases
- Security features with trusted origins

## Development

```bash
pnpm install
pnpm compile
pnpm start
```

## Build

```bash
pnpm build:linux  # Linux
pnpm build:win    # Windows
pnpm build:mac    # macOS
pnpm build:all    # All platforms
```
